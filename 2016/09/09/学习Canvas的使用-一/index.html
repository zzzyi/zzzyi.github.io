学习Canvas的使用(一)

2016-09-09

Canvas

##
Canvas 的获取方法有两种：

1.使用 Bitmap 创建一个位图,在这个位图上创建一个 Canvas ：
Bitmap b = Bitmap.createBitmap(100,100,Bitmap.Config.ARGB_8888);
Canvas c = new Canvas(b);
创建一个尺寸是100*100的 Bitmap，使之作为 Canvas 的操作对象。
2.通过重写 View.onDraw 方法，View 中的 Canvas 会被当做参数传递过来，我们可以直接在 View 中操作这个 Canvas 。 

/**   
 * 自定义一个简单的View   
 */
class CanvasView extends View{   

    Paint paint;   

    public CustomView1(Context context) {   
        super(context);   
        paint = new Paint(); //设置一个画笔   
        paint.setColor(Color.BLACK); //设置画笔的颜色位黑色  
        paint.setStrokeJoin(Paint.Join.ROUND); //设置连接处为圆弧  
        paint.setStrokeCap(Paint.Cap.ROUND); //设置画笔样式为圆形样式  
        paint.setStrokeWidth(1); //设置画笔的宽度为1  
    }   

    //重写onDraw方法，我们可以在这里通过Canvas绘制图形   
    @Override   
    protected void onDraw(Canvas canvas) {   

    }   

}

###简单应用

一、时钟

1、了解了基础图形，现在要针对这个图形进行绘制，首先对图形进行拆分，对该图形的绘制有一个大致的思路：
(1) 画表盘，即画一个圆，使用方法 Canvas.drawCircle(x,y,r,paint)；
(2) 在规定路径上绘制文本，使用方法 Canvas.drawTextOnPath (text,path,hOffset,vOffset,paint) ，基本图形中介绍了一种在路径上绘制文本的方法，但是路径为直线路径，而图形中为圆弧路径，所以要使用 Path.addArc() 的方法来添加圆弧路径；
(3) 绘制刻度，即画线，使用方法 Canvas.drawLine(startX,startY,endX,endY,paint);
(4) 绘制时针， 即画线、画圆，方法同上。
2、有了上述的方法，各个图形只是一堆零散的器件，无法按想象中的组合在一起，这个时候需要使用参考坐标，即中心点，在这个图形中很容易看出，
